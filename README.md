# 墨流し — sumi-nagashi

色を選んで水面に触れると、墨が落ち、にじみ、渦を巻いて静かに溶けていく
和風インタラクティブアートのサイト。

## 動かし方

ビルド不要の静的サイトです。ただし WebGL を使うため、`file://` ではなく
ローカルサーバー経由で開いてください。

```sh
npx http-server -p 8080
# → http://localhost:8080 を開く
```

GitHub Pages / Netlify / Vercel などにそのまま置いても動きます。

## 遊び方

- 下の盆から色(伝統色)を選ぶ
- **タップ / クリック** … 墨が一滴落ちる
- **押さえたまま動かさない** … 墨がゆっくり溶け出す
- **ドラッグ / スワイプ** … 水をかき混ぜる(マルチタッチ対応)
- 左上「水を替える」で真っさらに戻す

## 仕組み

- GPU(WebGL フラグメントシェーダ)上で安定流体法
  (Jos Stam, *Stable Fluids*)を解いています:
  移流 → 渦度強制 → 発散計算 → 圧力(Jacobi 反復)→ 勾配減算
- 墨は RGB の「吸収量」として保持し、表示時に
  `紙色 × exp(-吸収量)`(Beer–Lambert 則)で合成。
  本物の顔料のような重なり・濃淡になります
- 「にじみ」は染料テクスチャへの拡散パスで表現

## 調整どころ(`main.js` 冒頭の `config`)

| 項目 | 効き方 |
|---|---|
| `BLEED` | にじみの速さ。上げるとふわっと広がる |
| `VELOCITY_DISSIPATION` | 水の粘り。下げると渦が長く残る |
| `DENSITY_DISSIPATION` | 墨の消えやすさ。0 にすると永遠に残る |
| `CURL` | 渦の強さ |
| `SOAK_INK` | 長押しで溶け出す墨の量 |
